# AI Personal Finance Manager

A web-based finance manager built using Flask, SQLAlchemy and MySQL.

## Features

- User Authentication
- Income Tracking
- Expense Tracking
- Transaction History
- Delete Transactions
- Transaction Filters
- Dashboard
- MySQL Database
- Goals Tracking

## Technologies

- Python
- Flask
- SQLAlchemy
- Flask Login
- HTML
- CSS
- MySQL
